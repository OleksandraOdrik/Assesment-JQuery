# Interactive UI Playground

## Project Description
This project is a small interactive web application built using HTML, CSS, and JavaScript with jQuery. It demonstrates how web pages can respond to user interaction in real time.

The goal of this project was to practice DOM manipulation, event handling, and dynamic styling while creating a simple but visually appealing interface.


## Technologies Used
- HTML
- CSS
- JavaScript
- jQuery


## Features
- Add custom text dynamically to the page
- Clear all added content with one button
- Interactive buttons with toggle effects (glow, highlight, focus)
- Show/hide text using jQuery animations
- Change background color using user input
- Save selected color using localStorage
- Reset background color to default


## Setup Instructions
1. Download or clone the repository  
2. Open the file **Assesment.html** in your browser  


## Usage
- Enter text in the input field and click **"Add Text"**  
- Use **"Clear All"** to remove added messages  
- Try the buttons in Section 1 to see visual effects  
- Click **"Disappear"** to toggle text visibility  
- Enter a color (e.g. pink, blue, lightgreen) and press **Submit**  
- Click **Reset Color** to return to the default background
